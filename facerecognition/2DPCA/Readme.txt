Face Recognition using 2DPCA
=============================================
1. Download ORL face database on "Downloads" page and extract to the same folder with M functions and script.

2. Run gen_train_test_set.m to create training set and testing set.

3. Run gen_labelfile_orl to generate 'orl_label.txt' for testing.

4. Experiments
	4a. Run FR_2DPCA.m to calculate the accuracy.
	4b. Run TDPCA_REC.m to test image reconstructed
